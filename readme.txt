# Loan Management System

Welcome to the Loan Management System! This web-based application helps streamline the loan process, from application to repayment.

## Setup Instructions

1. **Download and install XAMPP:** [Download XAMPP](https://www.apachefriends.org/index.html)
2. **Install a text editor:** Use Notepad++ or Sublime Text 3.
3. **Download the source code:** [Download ZIP](#) <!-- Replace with your download link -->
4. **Extract and copy:** Extract the ZIP file and copy the "Loan_Management_System" folder.
5. **Paste in XAMPP directory:** Paste the folder inside the XAMPP root directory (e.g., `C:\xampp\htdocs`).
6. **Set up database:** Open PHPMyAdmin (http://localhost/phpmyadmin) and create a new database named `loan_db`. Import the `loan_db.sql` file located in the "SQL file" folder of the downloaded package.
7. **Run the application:** Navigate to http://localhost/Loan_Management_System in your web browser.
